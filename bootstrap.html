Introduction to Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Why Bootstrap?

  • Responsive design out of the box
  • Mobile-first approach
  • Extensive library of components
  • Easy to use and customize
  • Large community and ecosystem

Brief History

Bootstrap was developed by Mark Otto and Jacob Thornton at Twitter. It was initially called "Twitter Blueprint" and created as a framework to encourage consistency across internal tools. The first public release was on August 19, 2011.

Getting Started

Bootstrap CDNs

The easiest way to start using Bootstrap is by including it via a CDN.

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Creating Your First Page

Here's a basic template for a Bootstrap page:

<!DOCTYPE html> <html> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> <title>My Bootstrap Page</title> </head> <body> <div class="container"> <h1>Hello, world!</h1> </div> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script> </body> </html>

Best Practices

Responsive Design

  • Use Bootstrap's grid system to create responsive layouts
  • Choose appropriate breakpoints for your design
  • Test on multiple devices and screen sizes

Performance

  • Only include the components you need
  • Use CDN versions of Bootstrap to leverage browser caching
  • Minify CSS and JavaScript files in production

Accessibility

  • Use semantic HTML elements
  • Ensure proper color contrast ratios
  • Provide accessible alternatives for images and media

Real-World Example: Theme Switcher

This example demonstrates how to create a theme switcher using Bootstrap and JavaScript.

Code Example


<button id="themeToggle" class="btn btn-primary">Toggle Theme</button>

<script>
document.getElementById('themeToggle').addEventListener('click', function() {
    const body = document.body;
    if (body.getAttribute('data-bs-theme') === 'dark') {
        body.setAttribute('data-bs-theme', 'light');
    } else {
        body.setAttribute('data-bs-theme', 'dark');
    }
});
</script>
